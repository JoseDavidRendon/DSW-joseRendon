Propósito del sitio

Esta página web tiene como objetivo ofrecer información sobre el uso de la sintaxis y selectores CSS, así como información sobre la interacción humano ordenador. En ella se explicará brevemente la sintaxis y los selectores del lenguaje de diseño CSS

Introducción

CSS (Cascading Style Sheets o hojas de estilo en cascada) es un lenguaje de diseño gráfico utilizado para definir la presentación de un documento HTML.

CSS es fácil de aprender y comprender, puede ser aprendido por cualquier persona con un poco de conocimiento sobre HTML. La sintaxis es la forma correcta de escribir una instrucción, es decir, el orden en que las instrucción deben ser escritas, cada lenguaje cuenta con una, para el caso de CSS, consiste en un identificador y una (o múlitples) instrucciones, así mismo, la sintaxis de la instrucción consiste en una clave y un valor separados por dos puntos.

Por su parte, la interacción humano-ordenador consiste en la forma en que los usuarios interaccionan con los ordenadores y se centra en el diseño de interfaces eficaces que mejoren la experiencia del usuario y la comunicación con los ordenadores.
