Sintaxis y Selectores CSS

Sintaxis CSS

La sintaxis CSS se puede dividir en tres partes: sleectores, propiedades y valores.

  • Los selectores se usan para identificar un elemento HTML específico
  • Las propiedades se usan para especificar una característica del elemento, como el color, el tamaño o la posición.
  • Los valores se usan para especificar el valor de la propiedad que se quiere modificar

Los selectores CSS, como se mencionó anteriormente, son palabras clave o frases que se usan para identificar un elemento de HTML para modificarlo, por ejemplo, para cambiar el color de tood el texto en un documento, se puede usar el selector "h1" para seleccionar todos los titulos de primer nivel y aplicar una porpiedad de estilo como "color:red", tambien s epueden seleccionar por medio del ID o nombre de la clase.
ejemplo:

h1 {color:red}
donde h1 es el selector, y lo que se encuentra encerrado entre llaves es la declaración.

Los selectores pueden ser muy simples, como en el ejemplo anterior, o pueden ser más complejos, usando varias cláusulas y combinaciones.

Tanto la propiedad como el valor son sensibles a las mayúsculas en CSS. El par se separa por dos puntos, ":", y los espacios en blanco antes, entre ellos y después, pero no necesariamente de ellos, son ignorados.

Mdn Web Docs. (2020). Propiedades y Valores de CSS [Imagen]. Recuperado de Sintaxis - CSS | MDN

Las declaraciones son agrupadas en bloques, que es una estructura delimitada por una llave de apertura, "{", y una de cierre, "}". Los bloques en ocasiones pueden anidarse, por lo que las llaves de apertura y cierre deben de coincidir.
